Setup Guide

Activate the AI Plan Assistant

Five-minute walkthrough to turn on natural-language Q&A over your client plans. Live status checker below confirms when each step is complete.

Checking API endpoint status…
1

Create an Anthropic API key

~ 2 minutes
  1. Go to console.anthropic.com and sign in (or create an account with your work email).
  2. In the left sidebar click Settings → API Keys.
  3. Click the Create Key button. Name it something like pyle-planning-prod.
  4. Copy the key immediately — it starts with sk-ant-api03-…. You can't view it again after closing the dialog.
  5. Navigate to Plans & Billing → Add credits. Start with $10.
Open Anthropic Console →
💰 Cost check: each AI query runs ~3-5¢ using claude-sonnet-4-5. $10 of credits covers roughly 200-300 queries. Typical advisor use of 5-10 queries/client × 50 clients/year ≈ $10-25/yr.
2

Add the key to Vercel

~ 2 minutes
  1. Open vercel.com/dashboard and click into the Pyle Plan Builder project.
  2. Click Settings (top nav) → Environment Variables (left sidebar).
  3. Click Add New. Fill in:
    • Key (exact spelling, all caps):
      ANTHROPIC_API_KEY
    • Value: paste the sk-ant-api03-… key from Step 1
    • Environments: check all three (Production, Preview, Development)
  4. Click Save.
Open Vercel Dashboard →
3

Redeploy to pick up the new key

~ 1 minute

The key only activates after a fresh deploy. In Vercel:

  1. Click the Deployments tab.
  2. Find the top (most recent) deployment.
  3. Click the ⋯ menu on that row → Redeploy.
  4. Check "Use existing Build Cache", then click Redeploy.
  5. Wait ~30 seconds for the green Ready checkmark.
💡 Shortcut: if you're comfortable with git, just push any commit and Vercel auto-redeploys. Example: git commit --allow-empty -m "redeploy" && git push
4

Verify it works

~ 30 seconds

Click the button below to ping /api/ask-plan with a test payload. A green result means the key is wired up correctly and you're ready to use the assistant.

Once the test passes:

  1. Go to the main app and hard-refresh (⌘⇧R on Mac, Ctrl+Shift+R on Windows)
  2. Sign in, load a sample (Harrison recommended), click Generate Plan
  3. A gold "Ask your plan" button appears bottom-right
  4. Click it and ask any question — response appears in 2-5 seconds
5

Set a budget alert (optional but recommended)

~ 1 minute

Protect yourself from a runaway bill.

  1. Back in Anthropic console: Settings → Limits
  2. Set a $50/month soft limit (or whatever feels right)
  3. You'll get an email when you reach 75% of the limit
Open Anthropic Limits →

Troubleshooting

SymptomLikely cause & fix
Test above returns "Add ANTHROPIC_API_KEY" Either the key wasn't saved in Vercel, or the deploy was created BEFORE the key was added. Re-check the env var spelling is exactly ANTHROPIC_API_KEY, then force a fresh redeploy (Step 3).
Test returns 401 Unauthorized The API key string is wrong (truncated, extra spaces, or the key was regenerated/deleted in Anthropic console). Create a new key and repeat Steps 1-3.
Test returns 429 Rate Limit Very unusual — Anthropic accounts start with generous rate limits. Wait 60 seconds and try again.
Test returns "Insufficient credits" Add more credits in Anthropic console → Plans & Billing.
No chat button appears on the main app Generate Plan hasn't been clicked yet. The button only appears when a plan is loaded (by design — nothing to ask about otherwise).
Button appears but clicking it does nothing Hard-refresh the page (⌘⇧R). Browser may be serving old JS.
