Set CRON_SECRET (protects the daily Plaid sync)

~ 2 minutes

The daily balance sync at 06:00 UTC runs as a Vercel cron. We protect the endpoint with a shared secret so nobody else can trigger it.

I generated a fresh random secret for you — copy it, then paste it into Vercel as CRON_SECRET.

loading…
  1. Click Open Vercel Env Vars below.
  2. Click "Add New".
  3. Key: CRON_SECRET
  4. Value: paste the secret you copied above
  5. Environments: check Production, Preview, Development
  6. Click "Save".
Open Vercel Env Vars
Note: Vercel cron jobs require a Pro plan. You're on one cron currently, which is supported on Hobby (one daily cron). If you add more in the future, you'll need to upgrade.
